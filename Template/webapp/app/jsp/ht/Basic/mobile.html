<!--必备样式-->
$globalColors: (
  default:#6492FF,
  defaulttwo:#C9D9FF,
  red:#ff6666,
  blue:#069,
  white:white,
  date:#000000,
  green:#00CC00,
  gray:#cccccc,
  dark-gray:#666666,
  medium-grey:#999999,
  lightgray:#e4e4e4,
  shadow:#B3CAFF
);
@function colors($color){
    $names: map-keys($globalColors);
    @if index($names, $color){
        @return map-get($globalColors, $color);
    }
};
@function s($px) {
    @return($px/32)*1rem;
}

颜色选择使用方法

使用方法s(设计稿尺寸值)

@function s($px) { @return($px/32)*1rem; }

32为设计稿的尺寸宽度，除以10。 例如设计稿宽度为320px 则为： @return($px/32)*1rem;

用例：样式为btn的按钮，在设计稿中的宽度为 35px 使用rem换算则为: .btn{width:s(35)}
